실행 방법
================================================================================

모든 E2E 테스트 실행:

  $ npm run test:e2e -- --grep "라이선스 페이지"

특정 브라우저만 실행:

  $ npm run test:e2e -- --list

보고서 확인:
  $ npx playwright show-report
